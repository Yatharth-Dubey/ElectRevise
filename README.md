# 🪷 ElectRevise

Election Management System

## Tech Stack

Frontend:
- React
- Axios

Backend:
- Node.js
- Express

Database:
- MySQL

## Setup

Clone

```bash
git clone REPO_URL
```

Install

```bash
npm install
```

Create env

```bash
cp .env.example .env
```

Import DB

```bash
mysql -u root -p DATABASE_NAME < database_schema.sql
```

Run

```bash
npm start
```
